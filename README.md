# cursor_theme (Kokomi)

珊瑚宫心海主题，适用于linux xorg系统

### 使用方法

- 将Kokomi文件放到 /usr/share/icons/
- 设置光标主题为Kokomi
    参见 [https://wiki.archlinuxcn.org/wiki/光标主题](https://wiki.archlinuxcn.org/wiki/%E5%85%89%E6%A0%87%E4%B8%BB%E9%A2%98)
